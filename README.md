# ReCert Landing Page

This repository contains the official landing page for **ReCert**, a full-stack automated certificate lifecycle management system developed for the MS AI Innovation Hub Showcase.

## Overview

ReCert is designed to automate the monitoring, renewal, and validation of digital certificates using a state-based workflow and event-driven architecture.

This landing page serves as a central hub for:
- Project overview
- System architecture visualization
- Demo access
- Team information
- Technical stack

## Features

- Clean, modern UI for showcasing the ReCert system
- Embedded logical architecture diagram
- Quick access to demo video, GitHub repository, and team resources
- Mobile-friendly design for QR code access at live events

## Tech Stack

- HTML, CSS
- Hosted via Netlify
